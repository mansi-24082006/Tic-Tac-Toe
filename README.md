# ❌⭕ Tic-Tac-Toe Game

A fun and interactive **Tic-Tac-Toe** game developed using **HTML, CSS, and JavaScript**.  
This project demonstrates core JavaScript logic, DOM manipulation, and responsive UI design.

---

## 🌐 Live Demo

🔗 **Live Project Link:**  
👉 [Live Link](https://mansi-99914.github.io/Tic-Tac-Toe/)

## 🚀 Features

- ❌ Player X vs ⭕ Player O
- 🧠 Automatic win and draw detection
- 🔄 Restart game option
- ⚡ Instant result display
- 🎨 Clean and modern UI
- 📱 Responsive design
- ⚙️ Built with pure JavaScript (no libraries)

---

## 🛠️ Technologies Used

- **HTML5** – Game structure
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Game logic and interactions
